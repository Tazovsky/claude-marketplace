# Claude Marketplace

Claude Code plugin marketplace by Tazovsky.

## Plugins

| Plugin | Description |
|--------|-------------|
| [claude-rshiny-plugin](https://github.com/Tazovsky/claude-rshiny-plugin) | R Shiny application management skills for Claude Code |

## Installation

```bash
/plugin marketplace add Tazovsky/claude-marketplace
/plugin install claude-rshiny-plugin@tazovsky-marketplace
```
