# Claude Marketplace

Claude Code skills collection by Tazovsky.

## Skills

| Skill | Command | Description |
|-------|---------|-------------|
| **codex** | `/codex review`, `/codex brainstorm`, `/codex plan` | Use Codex CLI (OpenAI) as a second opinion for code review, brainstorming, and plan review |
| **rshiny-run** | `/rshiny-run "instructions"` | Launch R Shiny app, open browser, follow instructions |
| **rshiny-stop** | `/rshiny-stop` | Stop a running R Shiny app |
| **rshiny-logs** | `/rshiny-logs` | Show recent R Shiny server logs |
| **rshiny-status** | `/rshiny-status` | Check if R Shiny app is running |

## Installation

```bash
/install-plugin Tazovsky/claude-marketplace
```
